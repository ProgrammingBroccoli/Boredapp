Beste mannen van DCC!

Hierbij de Flutter uitdaging met de bored API.
Ik heb bij de implementatie weinig rekening gehouden met het scalen naar devices.
De app is geschaald op de grootte van de IPHONE 11 PRO MAX

Ik ben alle 3 de levels van de uitdaging aangegaan!

Level 1: implementatie van de API en opvragen van een random activity
level 2: Implementatie van een 2e scherm waar voorkeuren ingevuld kunnen worden, op basis daarvan zal er een activiteit opgevraagd worden.
level 3: om aan te tonen dat ik ook gemakkelijk met flutter packages werk heb ik bij het laden van de random activiteit (Openen van de app) een laadscherm toegevoegd
